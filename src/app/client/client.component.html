To create a new/custom component

1) Add a folder with that component name under app folder (as sub folder)

2) Add component_name.component.ts file


i)import Opening AngularBracket  Component Closing AngularBracket from '@angular/core' //here C of Component is capital


ii)build that component 

        with  
             selector : 'app-NewComponentName',
             templateUrl : './NewComponentName/NewComponentName.component.html' //give it the html component file URL
        

3) Need to export This Class

export class NewComponentNameComponent

4)Add Somthing to your HTML Page if this component 5) Now need to include this component so it will display on web page
